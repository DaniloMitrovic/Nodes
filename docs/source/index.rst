Welcome to FreeCAD Nodes documentation!
===================================

**FreeCAD Nodes** is a graphical algorithm editor for the parametric 3D modeler FreeCAD. By connecting function blocks
(so called nodes) in series or parallel, complex 3D models can be created, processed or animated.

Check out the :doc:`usage` section for further information, including the :ref:`installation` of the addon via
FreeCAD's Addon Manager.

.. note::
   This project is under active development.
